# React Native Book Explorer

A cross-platform mobile application for book readers built with **React Native** and **Expo**.  
The app connects to the **Google Books API** to search and explore books. Works on both **iOS** and **Android**.

## Features

- Search for books using Google Books API
- View book details (title, author, description, cover)
- Cross-platform support via React Native & Expo
- Simple and responsive UI
